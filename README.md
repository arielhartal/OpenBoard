# OpenBoard Monorepo

This repository currently hosts two related front-end applications:

- openboard/ – the original Create React App build of OpenBoard.
- insightboard/ – the new InsightBoard experience built with Next.js, TypeScript, and Tailwind.

Keep both directories for now while the migration is in progress. Once InsightBoard covers all functionality, the legacy app can be archived or removed. Each project maintains its own tooling, dependencies, and README inside its folder.
